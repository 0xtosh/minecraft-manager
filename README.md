# minecraft-manager

A lightweight web-based manager for the server config and mods directory with start/stop/stop+backup/restart functionality.

## Features

- Web-based interface for managing a Minecraft server
- Manage server config and mods directories
- Start, stop, backup, and restart server controls
- Lightweight setup and operation

## Getting started

1. Clone the repository.
2. Install dependencies.
3. Configure your server paths and settings.
4. Start the application.

## Usage

Use the web interface to manage your server configuration, mods, and lifecycle actions.
